\doxysubsection{Detailed Description}
Represents a circle in 2D space. 

The \mbox{\hyperlink{classCircle}{Circle}} class is derived from the \mbox{\hyperlink{classShape}{Shape}} class and defines a circle with a specified radius, center, fill color, stroke color, and stroke thickness.
