\doxysubsection{Detailed Description}
Represents a polygon in 2D space. 

The \mbox{\hyperlink{classPolygon}{Polygon}} class is derived from the \mbox{\hyperlink{classShape}{Shape}} class and defines a polygon with a variable number of vertices.
