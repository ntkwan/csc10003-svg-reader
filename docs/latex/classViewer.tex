\doxysubsection{Detailed Description}
Represents a viewer for handling events and interactions with an SFML window. 

The \mbox{\hyperlink{classViewer}{Viewer}} class is responsible for handling events, such as zooming, rotating, and dragging, to interact with an SFML window and view.
