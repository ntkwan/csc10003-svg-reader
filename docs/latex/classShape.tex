\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
The position of the specified point on the shape 
\end{DoxyReturn}
\begin{DoxyNote}{Note}
The returned point is in local coordinates, that is, the shape\textquotesingle{}s transforms (position, rotation, scale) are not taken into account. 

The result is undefined if index is out of the valid range. 

The number of points of the shape is defined by the concrete implementation. 

The returned vector is constant, which means that you can\textquotesingle{}t modify its coordinates when you retrieve it. 

This is a pure virtual method, so it has to be implemented by the derived class. 
\end{DoxyNote}
