\doxysubsection{Detailed Description}
Represents a rectangle in 2D space. 

The \mbox{\hyperlink{classRect}{Rect}} class is derived from the \mbox{\hyperlink{classShape}{Shape}} class and defines a rectangle with a specified width, height, position, fill color, stroke color, and stroke thickness.
