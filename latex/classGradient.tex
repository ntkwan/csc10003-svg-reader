\doxysubsection{Detailed Description}
A class that represents a gradient. 

The \mbox{\hyperlink{classGradient}{Gradient}} class is an abstract class that represents a gradient. It contains a vector of \mbox{\hyperlink{classStop}{Stop}} objects that represent the stops of the gradient. It also contains a pair of \mbox{\hyperlink{classVector2D}{Vector2D}} objects that represent the start and end points of the gradient.
