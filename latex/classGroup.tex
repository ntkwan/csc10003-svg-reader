\doxysubsection{Detailed Description}
A composite class that contains a vector of shape pointers (polymorphic). 

The \mbox{\hyperlink{classGroup}{Group}} class is derived from the \mbox{\hyperlink{classSVGElement}{SVGElement}} class and defines a group of SVGElements. The \mbox{\hyperlink{classGroup}{Group}} class is a composite class that contains a vector of \mbox{\hyperlink{classSVGElement}{SVGElement}} pointers (polymorphic). The \mbox{\hyperlink{classGroup}{Group}} class is used to group SVGElements together.
