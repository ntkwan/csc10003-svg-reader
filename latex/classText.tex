\doxysubsection{Detailed Description}
Represents text in 2D space. 

The \mbox{\hyperlink{classText}{Text}} class is derived from the \mbox{\hyperlink{classSVGElement}{SVGElement}} class and defines a text element with a specified position, string, fill color, and font size.
